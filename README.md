# Hello

This is a simple TypeScript project.

## Installation

```bash
npm init / bun init
```

```bash
npm install / bun install
```

## Usage

```bash
npm run src/index.ts / bun src/index.ts
```

### Endpoint
```bash
http://localhost:3000/hello
```

  "message": "Hello, World!"
